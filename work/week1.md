# 第 1 周执行清单

## 本周目标

本周只做一件事：

**把项目骨架、数据结构和文档入库主链路跑通。**

本周结束时，你至少要做到：

1. 项目目录已经搭好
2. 数据库表已经设计好
3. 可以上传文档
4. 可以解析文档并切块
5. chunk 和元数据可以写入 PostgreSQL
6. 有一版架构图和 README 大纲

## 当前实际进度

基于当前仓库代码，第一周已经完成到下面这个程度：

1. 项目骨架已经搭好
2. `knowledge_base` / `document` 两张核心表已经落地
3. PostgreSQL 已真实接通
4. Flyway 已真实执行
5. 知识库创建接口已完成
6. 文档上传接口已完成
7. 原始文件本地存储已完成
8. 文档去重校验已完成

当前还没完成的第一周内容：

1. Redis 连通
2. Markdown / txt / PDF 解析
3. 文本切块
4. `document_chunk` 入库
5. 架构图

所以第一周当前状态不是“未开始”，也不是“全部完成”，而是：

**已经完成上传入口和数据入库骨架，尚未进入解析与切块阶段。**
