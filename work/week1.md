# 第 1 周执行清单

## 本周目标

本周只做一件事：

**把项目骨架、数据结构和文档入库主链路跑通。**

本周结束时，你至少要做到：

1. 项目目录已经搭好
2. 数据库表已经设计好
3. 可以上传文档
4. 可以解析文档并切块
5. chunk 和元数据可以写入 PostgreSQL
6. 有一版架构图和 README 大纲
